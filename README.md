# music

基于网易云api音乐播放后端

这里使用的是[Binaryify/NeteaseCloudMusicApi: 网易云音乐 Node.js API service](https://github.com/Binaryify/NeteaseCloudMusicApi)的网易云api



运行前在musicServer\db\connection.js配置数据库



#### 安装

```
cd musicApi
npm install
cd ..
cd musicServer
npm install
```



#### 运行

```
cd musicApi
node app.js
cd ..
cd musicServer
node app.js
```
